📊 Marketing Campaign ROI Analysis (EDA)

Overview:

Conducted Exploratory Data Analysis (EDA) on a 200,000-record marketing dataset to analyze campaign effectiveness across different channels and support data-driven budget decisions.

Key Analysis:

ROI comparison across marketing channels

Evaluation of acquisition cost and click performance

Marketing funnel analysis (Impressions → Clicks → Conversions)

Monthly trend analysis of ROI

Performance assessment across customer segments

Key Insights:

Facebook and Website channels achieved comparatively higher ROI.

Email campaigns incurred higher acquisition costs with moderate returns.

Funnel analysis indicates opportunities to improve conversion efficiency.

ROI shows slight variation across months and customer segments.

Recommendation:

Focus on strengthening high-ROI channels while optimizing cost and performance in lower-performing campaigns to improve overall marketing efficiency.

Tools Used:

Python
Pandas
Matplotlib
Seaborn
Google Colab
